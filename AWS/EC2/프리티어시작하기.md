##`ssh로 ec2접속하기`
-AWS에서 발급 받은 pem 파일이 있는 위치로 이동합니다. 
>> cd "키 페어가 저장된 경로" (ex : cd desktop/aws)
-chmod 400 파일명.pem (여기서400은 나에게만`읽기`(4)권한 600은 나에게만`읽기+쓰기`(4+2)권한 입니다.)
>> $ chmod 400 "키 페어 파일 경로" (ex : chmod 400 test_server.pem)
-sudo ssh -i '파일명.pem' AMI@아이피주소
>> (ex : ssh -i test_server.pem ubuntu@12.12.12.123)
