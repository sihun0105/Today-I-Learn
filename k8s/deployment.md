#### 디플로이먼트 yaml작성법

```
apiVersion:
kind:
metadata:
    name: #디플로이먼트 이름
spec:
    selector: #셀렉터 설정
        machLabels: #셀렉터가 선택할 관리 대상 레이블
replicas: #레플리카 설정
template: #파드의 정보
    metadata: #파드의 메타데이터
    spec: #파드의 스펙
```
셀렉터 : 디플로이먼트가 특정한 레이블이 부여된 파드를 관리할 수 있도록 하는 설정이다.
레플리카 : 파드의 레플리카 즉 파드 수를 몇개로 유지할지 설정한다.
템플릿 : 생성할 파드의 정보를 기재한다. 다만 `파드이름`은 설정하지 않는다. 파드의 이름보다 레이블로 관리하는것이 보편적이다.
